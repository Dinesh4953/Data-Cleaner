# 🧹 Data Cleaner – Machine Learning Powered Data Cleaning Web App

Data Cleaner is a web application that helps users **upload, analyze, and clean datasets easily** through a simple and interactive dashboard. It allows users to explore dataset information and perform basic cleaning operations without writing code.

## 🚀 Features

- Upload CSV datasets
- View dataset information
- Interactive data table preview
- Data filtering and cleaning
- Simple dashboard UI
- Machine learning libraries for data processing
- Deployed web application

## 🛠 Tech Stack

- **Backend:** Django, Python
- **Data Processing:** Pandas, Scikit-learn
- **Database:** MongoDB (MongoEngine)
- **Frontend:** HTML, Bootstrap, JavaScript, jQuery
- **Deployment:** Render
- **Other Tools:** Plotly, Imbalanced-learn

## 📸 Project Demo

Upload a dataset and instantly:
- View dataset structure
- Explore columns and data types
- Filter and clean data
- View processed dataset in table format
